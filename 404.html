Hi, I'm Dave

I'm a gameplay programmer who's been working for Frontier Developments since June 2020, creating new features and expanding existing mechanics for the DLC of Planet Zoo. I've always been very passionate about Game Development, and here you can see a selection of projects I've worked on that demonstrate what skills I have to offer. I hope you find it interesting!

Professional Projects


Planet Zoo - Overview

A brief summary of my role and responsibilities as part of the Planet Zoo team. Mentions all the skills I've learned and utilized as a gameplay programmer.

Planet Zoo - Brachiation

A feature allowing animals to hang from and traverse between certain pieces put down by the player. The mechanic also interacts with the game's existing climbing system.

Planet Zoo - Burrows

A feature allowing animals to rest, seek shelter, and give birth underground. This feature required large AI changes to get a large new set of behaviours, all while requiring no new animations.

Personal Projects


Volumetric Cloud Experiments

A dissertation investigating the performance of various rendering techniques for clouds, including the creation of an experimental tool implementing raymarching.

Realtime Multiplayer Implementation

Adding online multiplayer to a previously worked on project in a self-produced game engine, also including physics and AI agents.

Paris Graphics Scene

A program demonstrating multiple diffent graphical techniques to render a Paris based landscape in OpenGL. Elements of the scene are built up over time, as the camera gives a guided tour.

Generative Adversarial Network Game

A project based on finding whether game content generation could be performed by machine learning methods, rather than typical procedural generation.
