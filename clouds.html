Volumetric Cloud Experiments

This was an individual project created to showcase many graphics techniques, while creating a landscape that built up over time, Game of Thrones title sequence style.

Key Skills

  • C++
  • OpenGL
  • GLSL

Here's just some of the techniques at play (more are mentioned in the video):

  • The eiffel tower is built using 5 seperate meshes, that each use a fragment shader that only renders a certain ammount of the mesh based on the time.
  • Postprocessing is present in the form of Bloom and Depth of Field effects.
  • A portal is present that renders a different scene. Scene management techniques allow this other scene to be drawn then presented only through the portal to the player.
  • Grass in the daylight scene is generated by tessellating a quad, and using a geometry shader to add a player facing quad on each tessellated vertex. These quads are textured and moved based on time to create a wind effect.

The program can run at a stable 60FPS, and allows the user to switch between free movement and the guided camera track, made using catmull rom splines.

What I'd do next time

Polish the graphics like crazy. The reflective water would benefit from a displacement map and a bit of transparency, and could also be tessellated to produce wave-like effects. The program could also benefit from cascaded shadow mapping, considering how blocky some of the shadows appear. I'd also like to improve the portal effect, to allow the transition to be seamless (the flash transition is only there to somewhat hide the fact that lights and shadows aren't present on the other side of the portal). This would require restructuring how the scene is rendered (rendering the portal scene completely before any rendering on the current scene, including transparency), but I reckon it'd be worth it.
