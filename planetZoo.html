Planet Zoo - Overview

I've been a part of the Planet Zoo team since August 2020, and in that time I've done a great deal. From animal behaviour code to experience with our render system, the information below provides a brief overview of my accomplishments, and how I've contributed to the 9 DLC releases I've worked on.

Summary

I've been a gameplay programmer during 9 DLC releases for Planet Zoo, a simulation game allowing players to build and manage a zoo. I'm currently the primary coder responsible for animal behaviour, obviously a key part of a game with a main gameplay loop of building habitats for animals to interact with.

My core responsibilities include:

  • Creating and maintaining the majority of animal AI code
  • Implementing the majority of new shader code additions
  • Familiarising new graduates with Frontier's Cobra engine
  • Communicating with animation to ensure animations are technically viable
  • Coordinating with production and design to ensure planned features are correct in scope

AI Example: I created a system enabling animals to dig holes on soft terrain, creating a tunnel network within a habitat.

Key Skills

  • C++
  • Lua
  • HLSL
  • SQL
  • In-House Engine Usage
  • Online Programming
  • Multi-Threaded Programming
  • Profiling and Debugging Tool Usage
  • Interdepartmental Communication
Additionally, I'm regularly chosen as a programming representative by our community team for livestreams and other social media, and I've also participated in various school outreach events for Frontier.

Graphics Example: I implemented various colour morph genetics, giving animals rare coats that could be passed down to their offspring.
