Planet Zoo - Overview

I've been a part of the Planet Zoo team since August 2020, and in that time I've done a great deal. From animal behaviour code to experience with our render system, the video below provides a brief overview of my accomplishments, and how I've contributed to the 8 DLC releases I've worked on.

  • Python
  • Node.js
  • HTML
  • CSS

This was a team project. I was responsible for the user interface, and the machine learning used to categorise errors and make stock predictions. We were tasked by Deutsche Bank, along with 28 other groups, to create a piece of software allowing the employees of Deutsche Bank to scour the stock market for anomalous data, helping to detect errors, and potential fraud. Using an agile development methodology, we developed a responsive web app that was praised for its ease of use, and comprehensive set of features. For those reasons, our solution ended up being chosen as the winner out of all 29 groups.
