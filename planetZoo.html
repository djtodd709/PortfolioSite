Summary

I've been a gameplay programmer during 12 DLC releases for Planet Zoo, a simulation game allowing players to build and manage a zoo. I'm the primary coder responsible for animal behaviour, obviously a key part of a game with a main gameplay loop of building habitats for animals to interact with.

My core responsibilities include:

  • Creating and maintaining the majority of animal AI code
  • Implementing the majority of new shader code additions
  • Familiarising new graduates with Frontier's Cobra engine
  • Communicating with animation to ensure animations are technically viable
  • Coordinating with production and design to ensure planned features are correct in scope

I've touched many aspects of the game, and created a lot of features I'm very proud of. Here's a selection of my highlights while working on Planet Zoo.

Natural Brachiation

AI/Animation Programming

A system allowing animals to swing around on freely placed scenery pieces.

  • Integrated into the pre-existing climbing system
  • Implemented code-driven locomotion between objects
  • Node based pathfinding used to find swinging routes
  • Implemented IK system for holding angular beams
  • Involved leading weekly inter-department meetings

Burrowing System

AI Programming

A system enabling animals to enter burrows to perform certain actions.

  • Also allowed tunnels to be dug on soft terrain, creating a tunnel network within a habitat
  • Added behaviour tree code for all actions performed underground
  • Created technical design that removed the need for additional animation work
  • Implemented camera code to view underground animals via an in-game webcam

Colour Morph Genetics

Graphics Programming

Implemented various colour morph genetics, giving animals rare coats that could be passed down to their offspring.

Presenting at EGX

Communication and Outreach

Presented information on animal behaviours, and the technology used to achieve graphical effects on animals from the Oceania Pack.
